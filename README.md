# Svalinn

Secure token generation and decoding.

## Installation

The package can be installed by adding `svalinn` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:svalinn, "~> 0.0.1"}
  ]
end
```

The docs can be found at [https://hexdocs.pm/svalinn](https://hexdocs.pm/svalinn).
